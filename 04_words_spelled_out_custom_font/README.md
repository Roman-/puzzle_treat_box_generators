# Words Spelled Out Puzzle

This generator creates a puzzle where a 4 or 5 digit code is spelled out using custom, tricky-to-read fonts.

![Puzzle Preview](use_this_pic.png)

## Features
- **Custom Fonts**: Choose from several unique fonts.
- **Legend**: Includes a legend mapping digits to words.
- **Customization**: Adjust colors, sizes, spacing, and more.
- **Randomization**: Quickly generate random puzzles.
