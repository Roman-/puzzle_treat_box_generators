# Interpretation 1: Small Symmetry Grids

## Concept

Each digit panel shows a small grid (3×3 or 4×4) with a symmetry-based pattern. One cell is marked with "?" and the child must identify the symmetry axis/rule to determine the missing color.

## Visual Layout

```
┌─────────────────────────────────────────────────────────┐
│  [Color Legend Bar: 0=Red, 1=Blue, 2=Green, ...]        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ┌───────┐   ┌───────┐   ┌───────┐   ┌───────┐         │
│  │ R │ B │   │ G │ G │   │ Y │ O │   │ P │ A │         │
│  ├───┼───┤   ├───┼───┤   ├───┼───┤   ├───┼───┤         │
│  │ ? │ B │   │ ? │ G │   │ O │ ? │   │ ? │ P │         │
│  └───────┘   └───────┘   └───────┘   └───────┘         │
│  Panel 1     Panel 2     Panel 3     Panel 4           │
│  (V-Mirror)  (H-Mirror)  (Diagonal)  (V-Mirror)        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## Pattern Types Used

1. **Vertical Mirror**: Left/right halves are identical
2. **Horizontal Mirror**: Top/bottom halves are identical
3. **Diagonal Mirror**: Diagonal axis symmetry
4. **Point Symmetry**: 180° rotational symmetry around center

## Grid Sizes

- 2×2 grid: Simplest, good for younger children
- 3×3 grid: More cells to establish pattern
- 4×4 grid: More complex, harder to spot symmetry

## Solving Mechanics

1. Child looks at panel
2. Identifies which cells are filled with colors
3. Notices the symmetry pattern
4. Determines what color must be in the "?" cell based on its symmetric counterpart
5. Looks up color in legend to get the digit

## Cell Display Options

- **Color only**: Just colored squares
- **Digit only**: Numbers shown in cells
- **Color + Digit**: Colored squares with white digit overlay

## Parameters

- Grid size (2×2 to 4×4)
- Pattern types to include
- Cell display mode
- Panel spacing
- Cell size
